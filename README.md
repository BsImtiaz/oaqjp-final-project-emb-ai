# Final project

## Emotion Detection Application

This project is an AI-based Emotion Detection web application developed using Python, Flask, and Watson NLP.

The application detects emotions such as:

- Anger
- Disgust
- Fear
- Joy
- Sadness

It also identifies the dominant emotion from the given text input.

## Technologies Used

- Python
- Flask
- Watson NLP
- HTML
- JavaScript

## Features

- Emotion Detection
- Dominant Emotion Identification
- Error Handling
- Unit Testing
- Flask Web Deployment
- Static Code Analysis
